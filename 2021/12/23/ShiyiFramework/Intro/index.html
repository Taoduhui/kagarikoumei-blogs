# 介绍

# 简介

ShiyiFramework 是一个微信小程序的简易封装，是为了使微信小程序与 typescript 更加贴合而开发出来的一套工具。
ShiyiFramework 通过使用 ShiyiAsm 对模板代码进行渲染，提供了绝对路径引用、双向绑定等功能。

# ShiyiAsm

ShiyiAsm 提供了对模板代码的渲染；以及页面模板、伪组件模板的创建；vscode、微信开发者工具的初始化功能。

# ShiyiPage

ShiyiPage 是 ShiyiFramework 对微信小程序 Page 对象的封装，提供了一系列数据绑定以及事件等功能。

# ShiyiPesudoComponent

ShiyiPesudoComponent 是 ShiyiFramework 提供的微信小程序 typescript 的组件支持。
