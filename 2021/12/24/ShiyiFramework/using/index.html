# 创建页面

在终端中执行 ShiyiAsm.exe -page [页面名] --overwrite/--skip 命令

将会在终端工作目录创建一个 ShiyiPage 模板

执行以上命令，将会创建出如下目录

创建完成后，需要在 Config/RouterConfig.ts 中配置相应的路由才可正常跳转，具体参考 路由

# 创建伪组件

在终端中执行 ShiyiAsm.exe -comp [组件名] --overwrite/--skip 命令

将会在终端工作目录创建一个 ShiyiPesudoComponent 模板

执行以上命令，将会创建出如下目录
